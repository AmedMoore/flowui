## Demos

<Column gap={4} customClassName={"mb-8"}>
    <Skeleton width="25rem" height="2rem" />

To make a skeleton like this, your code will be looks like the following one.

```javascript
import { Skeleton } from '@flowui/react/feedback';
import { Column } from '@flowui/react/layout';

export const Skeletons = () => {
    return (
        <Column>
            <Skeleton width="25rem" height="2rem" />
            <Skeleton width="40rem" />
            <Skeleton width="35rem" />
            <Skeleton width="40rem" />
            <Skeleton width="35rem" />
        </Column>
    );
};
```

## API

### Import

```javascript
import { Skeleton } from "@flowui/react";
// or
import { Skeleton } from "@flowui/react/feedback";
// or
import Skeleton from "@flowui/react/feedback/skeleton";
```

### Props

| Name | Type | Required | Default | Description |
| ---- | ---- | -------- | ----------- | ------- |
| width | number - string | false  | "100%" | A prop to control the skeleton width |
| height | number - string | false | "0.625rem" | A prop to control the skeleton height |
