## API

### Import

```javascript
import { Chip } from "@flowui/react";
// or
import { Chip } from "@flowui/react/data";
// or
import Chip from "@flowui/react/data/chip";
```

## Types

### Props

| Name  | Type  | Required | Default | Description                                  |
| ----- | ----- | -------- | ------- | -------------------------------------------- |
| color | Color | false    | "basic" | property to change the chip background color |
| size  | Size  | false    | "md"    | property to change the Chip element size     |

<br />

### Size

```ts
type Size = "auto" | "xs" | "sm" | "md" | "lg" | "xl";
```

### Color

```ts
type Color =
  | "basic"
  | "white"
  | "black"
  | "primary"
  | "secondary"
  | "success"
  | "warning"
  | "error"
  | "gradient";
```
