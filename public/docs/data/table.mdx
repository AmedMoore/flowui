## API

### Import

```javascript
import { Table } from "@flowui/react";
// or
import { Table } from "@flowui/react/data";
// or
import Table from "@flowui/react/data/table";
```

## Types

### Props

| Name    | Type           | Required | Default   | Description                                           |
| ------- | -------------- | -------- | --------- |------------------------------------------------------ |
| columns | ColumnData[]   | true     | undefined | property to change the chip background color          |
| data    | TData[]        | true     | undefined | prop to control the data that you want to render it   |
| caption | string         | false    | undefined | property to change the Chip element size              |
| shadow  | true - Shadow  | false    | undefined | property to change the Table shadow                   |

<br />

### ColumnData
```tsx
type ColumnData<TData extends Record<string, unknown>> = {
  label: string;
  dataKey: keyof TData;
  width?: number;
  colSpan?: number;
  format?: (value: TData[keyof TData]) => string;
  component?: (props: React.PropsWithChildren) => React.ReactNode;
};
```

### Shadow
