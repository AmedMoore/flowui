# Button Group

> Button Group is an awesome component that helps you combine a group of buttons instead of using another div and some custom CSS. Sounds good? Let's see some examples using all variants and colors. I Hope it fulfils your needs ðŸ˜ƒ.

## Demos

### Variants

Solid Group

<ButtonGroup variant={"solid"}>
    <Button>One</Button>
    <Button>Two</Button>
    <Button>Three</Button>
</ButtonGroup>

<br />

```javascript
import { Button, ButtonGroup } from "@flowui/react/basic";

export const SolidGroup = () => {
    return (
        <ButtonGroup variant={"solid"}>
            <Button>One</Button>
            <Button>Two</Button>
            <Button>Three</Button>
        </ButtonGroup>
    );
}
```

## API

### Import

```javascript
import { ButtonGroup } from "@flowui/react";
// or
import { ButtonGroup } from "@flowui/react/basic";
// or
import ButtonGroup from "@flowui/react/basic/button-group";
```

### Props

| Name         | Type              | Required  | Default     | Description                                                                                                            |
| ------------ | ----------------- |  -------- | ----------- | ---------------------------------------------------------------------------------------------------------------------- |
| size         | Size              | false     | "md"        | Button size for the whole div                                                                                                       |
| color        | Color             | false     | "primary"   | Button color prop                                                                        |
| variant      | ButtonVariant     | false     | "text"      | same as button prop                                                                  |
| iconOnly     | boolean           | false     | false       | this prop help you to make it act as icons only                   |
| disabled     | boolean           | false     | false       | disabled prop is for controlling the disable state for the button                                                      |
| loading      | boolean           | false     | false       | control the loading state |
| width        | ButtonWidth       | false     | "auto"      | Button size: "auto" or "full"                                                                                          |
| children     | React.ReactElement - React.ReactElement[] - null - false  | false     | "none"      | The ButtonGroup children that will be wrapped and sent as a children. |


## Types

### Size

```typescript
type Size = "auto" | "xs" | "sm" | "md" | "lg" | "xl";
```

### Color

```typescript
type Color = "basic" | "primary" | "secondary" | "success" | "warning" | "error" | "gradient";
```

### Button Variant

```typescript
type ButtonVariant = "solid" | "bordered" | "flat" | "text";
```

### Button Width

```typescript
type ButtonWidth = "auto" | "full";
```
