# Button

> Button component helps you make a modern Button with a few props. Please give it a shot.

## API

### Import

```javascript
import { Button } from "@flowui/react";
// or
import { Button } from "@flowui/react/basic";
// or
import Button from "@flowui/react/basic/button";
```

### Props


| Name         | Type              | Required  | Default     | Description                                                                                                            |
| ------------ | ----------------- |  -------- | ----------- | ---------------------------------------------------------------------------------------------------------------------- |
| as           | React.ElementType | false     | "button"    | Changes which tag component outputs                                                                                    |
| variant      | ButtonVariant     | false     | "solid"     | you can change the variant of button using this prop.                                                                  |
| color        | Color             | false     | "primary"   | update the button color depending on Color type                                                                        |
| size         | Size              | false     | "md"        | button size prop                                                                                                       |
| width        | ButtonWidth       | false     | "auto"      | Button size: "auto" or "full"                                                                                          |
| shadow       | boolean           | false     | "none"      | this prop let's you have an awesome shadow beyond the button                                                           |
| type         | ButtonType        | false     | "button"    | button type: "submit" or "button"                                                                                      |
| disabled     | boolean           | false     | false       | disabled prop is for controlling the disable state for the button                                                      |
| loading      | boolean           | false     | false       | button component provides a loading icon spinner that indicates the state of button you can control it using this prop |
| iconOnly     | boolean           | false     | false       | it makes the button act as icon only. so you can wrap any icon with the button and it'll act as icon                   |
| circle       | boolean           | false     | false       | it makes the button have a circle shape                                                                                |
| focusable    | boolean           | false     | true        | this property gives a border to the button when it's focused                                                           |
| leadingIcon  | JSX.Element       | false     | "undefined" | to have a icon in the beginning of button send the icon element in this prop                                           |
| trailingIcon | JSX.Element       | false     | "undefined" | to have a icon in the end of button send the icon element in this prop                                                 |


```typescript
type Color = "basic" | "primary" | "secondary" | "success" | "warning" | "error" | "gradient";
```

### Size

```typescript
type Size = "auto" | "xs" | "sm" | "md" | "lg" | "xl";
```

### Button Width

```typescript
type ButtonWidth = "auto" | "full";
```

### Button Type

```typescript
type ButtonType = "button" | "submit";
```
