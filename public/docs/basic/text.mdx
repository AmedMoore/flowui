## API

### Import

```javascript
import { Text } from "@flowui/react";
// or
import { Text } from "@flowui/react/basic";
// or
import Text from "@flowui/react/basic/text";
```


### Props


| Name      | Type              | Required  | Default | Description                                                          |
| --------- | ----------------- |  -------- | ------- | -------------------------------------------------------------------- |
| as        | React.ElementType | false     | "p"     | Changes which tag component outputs                                  |
| size      | SizeText          | false     | "none"  | Several options to change the font size like: xs, sm, lg, xl, etc... |
| color     | Color             | false     | "none"  | A property to change the font color                                  |
| weight    | FontWeight        | false     | "none"  | A property to change the font weight                                 |
| align     | TextAlignment     | false     | "none"  | Change the font alignment                                            |
| transform | TextTransform     | false     | "none"  | Change the font transform-property from several options              |
