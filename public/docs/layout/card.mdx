## API

### Import

```javascript
import { Card } from "@flowui/react"
// or
import { Card } from "@flowui/react/layout"
// or
import Card from "@flowui/react/layout/card"
```

## Types

| Name         | Type              | Required  | Default     | Description                                                                                  |
| ------------ | ----------------- | --------- | ----------- |----------------------------------------------------------------------------------------------|
| fullWidth    | boolean           | false     | "none"      | card width size: true or false                                                               |
| shadow       | boolean - Shadow  | false     | "none"      | this prop let's you have an awesome shadow beyond the table                                  |
| bordered     | boolean           | false     | true        | this prop let's you make a border to the table                                               |
